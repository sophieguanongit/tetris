# Tetris Skill

经典俄罗斯方块，单文件 HTML 游戏，浏览器打开即玩。

## 调用方式

直接将 HTML 文件发送给用户：

```bash
send_file /root/.openclaw/skills/tetris/tetris.html
```

用户收到后在浏览器打开即可游玩。

## 游戏操作

| 按键 | 功能 |
|------|------|
| ← → | 左右移动 |
| ↑ | 旋转 |
| ↓ | 软降（+1分） |
| Space | 硬降（+2分/格） |
| P | 暂停/继续 |
| R | 游戏结束后重新开始 |

## 游戏规则

- 经典 7 种方块（I O T S Z J L）
- 消 1/2/3/4 行分别得 100/300/500/800 × 等级
- 每消 10 行升一级，下落速度递增
- 方块堆到顶部游戏结束

## 输出

在浏览器中打开游戏页面，用户直接用键盘操作。
